# 📌 Project Overview

This project transforms raw Swiggy sales data into meaningful business insights through an interactive and visually appealing Excel dashboard.

The dashboard helps analyze:

✅ Sales Performance  
✅ Customer Ratings  
✅ Order Trends  
✅ State-wise Sales Distribution  
✅ Food Category Analysis  
✅ Top Performing Cities  

---

# 🎯 Business Problem

Food delivery platforms generate huge amounts of sales data daily. Businesses need data-driven insights to:

- Improve customer satisfaction 😊
- Increase operational efficiency ⚙️
- Identify top-performing regions 🌍
- Track sales growth 📈
- Optimize decision-making 🎯

This dashboard provides a centralized view of important business metrics and trends.

# 🔍 Insights Generated

📌 Bengaluru generated the highest sales revenue.

📌 Non-Veg orders contributed the majority of sales.

📌 Weekend sales were comparatively higher than weekdays.

📌 Some states showed high order volume but lower customer ratings.

📌 Average order value remained stable across months.
